# Handoff Checklist

> Use this at the end of every session.

- [ ] **current-state.md** updated with:
  - [ ] Date
  - [ ] Session summary (what was done)
  - [ ] Active branch
  - [ ] Build status
  - [ ] Blockers (or "None")
- [ ] **next-steps.md** updated with:
  - [ ] Completed items removed
  - [ ] New items added
  - [ ] Priority order reflects current state
- [ ] **decisions.md** updated (if any non-obvious decision was made)
- [ ] All work committed to git
- [ ] No uncommitted secrets or credentials
