Support

Get help with SimmerSmith

SimmerSmith is an Apple-first, self-hosted meal planning app. If something is broken, confusing, or missing, send a note and include enough detail to reproduce the issue.

Contact

Email: support@finklea.dev

Secondary support channel: GitHub Issues

Do not send bearer tokens, database files, or private household data unless you are asked to share a redacted sample.

What to include in a bug report

  • The app name: SimmerSmith
  • What you were trying to do
  • What happened instead
  • The app version or commit/build if you know it
  • Your device model and iOS version
  • Whether you were connected to a local or remote SimmerSmith server
  • Screenshots or screen recordings if they help explain the issue

Common support topics

  • Connecting the iPhone app to a SimmerSmith server
  • Recipe import from URL, photo, camera scan, or PDF
  • Sync issues between the app and the configured server
  • Grocery state, exports, or week-planning data that looks incorrect

For privacy questions, use support@finklea.dev and include “Privacy” in the subject line.
