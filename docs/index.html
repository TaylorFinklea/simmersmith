SimmerSmith

SimmerSmith is a self-hosted meal planning app built around recipes, weekly planning, groceries, and household food workflows. The current product combines a native iPhone app, a FastAPI service layer, SQLite persistence, and a lightweight browser admin surface.

Get Support Read Privacy Policy

What it does

Browse and edit recipes, import recipes from URLs or scans, review the current week, manage groceries, and sync with a user-configured SimmerSmith server.

How it works

The iPhone app keeps a local cache for offline reading, while the configured server remains the source of truth for recipes, week plans, grocery state, and exports.
